# Trabajo-Practico-Integrador-TPI-Programaci-n-1
# Gestión de Datos de Países en Python

## Descripción

Este proyecto fue desarrollado como Trabajo Práctico Integrador de Programación 1.

La aplicación permite gestionar información de países almacenada en un archivo CSV mediante el uso de listas, diccionarios, funciones, estructuras condicionales y repetitivas.

El sistema permite agregar, buscar y actualizar países, aplicar filtros y ordenamientos sobre los datos, y generar estadísticas generales sobre la información almacenada.

---

## Requisitos

* Python 3.x
* Archivo `paises.csv` ubicado en la misma carpeta que el programa

---

## Estructura de los datos

Cada país contiene la siguiente información:

* Nombre
* Población
* Superficie
* Continente

Ejemplo:

```csv
nombre,poblacion,superficie,continente
Argentina,45376763,2780400,America
Brasil,213993437,8515767,America
Alemania,83149300,357022,Europa
Japon,125800000,377975,Asia
```

---

## Funcionalidades

### Gestión de países

* Agregar un país.
* Buscar un país por nombre.
* Actualizar población y superficie de un país.

### Filtros

* Filtrar países por continente.
* Filtrar países por rango de población.
* Filtrar países por rango de superficie.

### Ordenamientos

* Ordenar países por nombre.
* Ordenar países por población.
* Ordenar países por superficie.
* Orden ascendente o descendente según corresponda.

### Estadísticas

* País con mayor población.
* País con menor población.
* Promedio de población.
* Promedio de superficie.
* Cantidad de países por continente.

### Validaciones

* No se permiten campos vacíos.
* Validación de datos numéricos.
* Control de opciones inválidas del menú.
* Control de rangos inválidos en filtros.
* Manejo de búsquedas sin resultados.
* Control de errores de formato en el archivo CSV.
* Mensajes claros de éxito y error.

---

## Ejecución

1. Descargar o clonar el repositorio.
2. Verificar que el archivo `paises.csv` se encuentre en la carpeta del proyecto.
3. Ejecutar el programa:

```bash
python TPI (Programación).py
```

4. Utilizar el menú interactivo para acceder a las distintas funcionalidades.

---

## Menú principal

```text
1. Agregar país
2. Buscar país
3. Actualizar país
4. Filtrar por continente
5. Filtrar por población
6. Filtrar por superficie
7. Ordenar por nombre
8. Ordenar por población
9. Ordenar por superficie
10. Mostrar estadísticas
11. Salir
```
